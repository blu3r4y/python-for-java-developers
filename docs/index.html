Why Python?¶

  • a mature, general-purpose, object-oriented, dynamically-typed programming language (Java is statically-typed)
  • released in 1991 (Java was released in 1995)
  • one of the dominating languages in machine learning, statistics, and data analytics
  • popular alternative to MATLAB or Octave
  • gaining in popularity year after year

(c) https://insights.stackoverflow.com/trends

Comparison to Java¶

Similarities¶

  • general-purpose, object-oriented, garbage-collected, large standard library

Differences¶

  • dynamically-typed, and not strongly-typed (type checks will only happen at runtime)
  • interpreted with CPython, and not just-in-time compiled with a JVM ("more flexibility, but slower execution times")
  • interactive mode provided out of the box ("you are looking at it right now!")

A few more differences ...¶

  • not that suitable for multi-threaded programming
  • no access modifiers like private or public ("but, there is a convention on how to mark code as private)"
  • no interfaces, no generics ("but, dynamically-typed languages have a solution for that")
  • no method overloading
